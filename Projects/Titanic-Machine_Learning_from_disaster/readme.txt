Titanic - Machine Learning from Disaster
Descripción
Este proyecto utiliza datos de pasajeros del Titanic para predecir si sobrevivieron al desastre, aplicando técnicas de Machine Learning. La competencia forma parte de un reto de Kaggle, ideal para aprender los fundamentos de procesamiento de datos y modelado predictivo.

Objetivo
Construir un modelo que prediga la supervivencia de los pasajeros del Titanic, utilizando datos como edad, sexo, clase, entre otros.

Requisitos
Python 3.x
Bibliotecas: pandas, numpy, scikit-learn, matplotlib, seaborn
Pasos
Cargar y explorar los datos.
Preprocesar los datos para el modelo.
Entrenar modelos de Machine Learning.
Evaluar y comparar los resultados.
